# Sales Dashboard Project (Python + Power BI)

## Project Overview
This project analyzes sales data to generate insights using Python and Power BI. It includes data cleaning, KPI calculations, visualization, and an Excel file prepared for dashboard creation.

---

## Objectives
- Clean and preprocess sales data
- Generate KPIs (Total Sales, Orders, Average Sales)
- Analyze sales by Category, Region, and Time
- Build visual insights using Python
- Create Power BI-ready Excel file

---

## Dataset Features
- Order ID
- Order Date
- Ship Date
- Category
- Sub-Category
- Region
- Sales

---

## Key Insights
- Total Sales and Order Count
- Top-performing Category
- Regional performance comparison
- Monthly sales trend
- Best Sub-Categories

---

## Visualizations
- Sales by Category (Bar Chart)
- Sales by Region (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Top 10 Sub-Categories (Horizontal Bar)

---

## Tools Used
- Python (Pandas, Matplotlib)
- Excel
- Power BI

---

## Output Files
- sales_dashboard.xlsx (Power BI-ready)
- Cleaned dataset CSV
- Visual charts

---

## How to Run

pip install pandas matplotlib openpyxl
python scripts/analysis.py
